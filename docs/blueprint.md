# **App Name**: Altitude

## Core Features:

- Project Creation & Management: Create, edit, and archive projects with detailed descriptions, types, and team assignments.
- Issue Tracking: Create, assign, prioritize, and track issues within projects.  Support various issue types (Task, Bug, Story) and custom fields.
- Kanban/Scrum Boards: Visualize workflows and manage issues using drag-and-drop Kanban and Scrum boards.
- Backlog Management: Plan and prioritize work items in a product backlog.  Refine backlog items, estimate effort, and assign them to sprints.
- Sprint Planning & Execution: Plan, start, and complete sprints.  Track sprint progress with burndown charts and velocity reports.
- Real-time Notifications & Activity Logs: Inform the team of project and issue activity, such as issue creation, status changes, comments, and assignments, providing a clear audit trail.
- AI-Powered Issue Summarization: Generates a concise summary of issue details, comments, and activity using a generative AI tool. Useful for quickly understanding issue status.

## Style Guidelines:

- Primary color: Deep blue (#2E3D83) to convey professionalism and trust.
- Background color: Light gray (#F0F2F5) to provide a clean and neutral canvas.
- Accent color: Vibrant purple (#A369D2) to highlight interactive elements and call-to-actions.
- Body and headline font: 'Inter', a grotesque-style sans-serif known for its clean and neutral appearance.
- Use a set of consistent and minimalist icons for project types, issue priorities, and other key actions.
- Employ a clear visual hierarchy with comfortable spacing, rounded corners (rounded-xl), and subtle shadows to create a modern, enterprise SaaS feel.
- Subtle micro-animations to indicate activity and communicate changes in state during drag and drop or on opening modals.
